# 🇿🇲 ZAMBIA AI

**Your AI Assistant from Zambia | Powered by DeepSeek R1 via OpenRouter**

Zambia AI is a free web app that helps Zambians with farming, business, education, and everyday questions. 
No credit card needed. Works on any phone with internet.

## ✨ Features
- **Smart Chat** - Ask anything: farming, business ideas, school, math
- **Fast Answers** - Powered by DeepSeek R1. More powerful than Gemini 2.0 Flash
- **Mobile Friendly** - Works on MTN, Airtel, Zamtel
- **Free Forever** - No API key needed from users. No login required
- **Zambian Focused** - Answers about Lusaka, maize farming, kwacha, etc.

## 🚀 How to Use
1. Open the app: [https://simwingachristopher0-ship-it.github.io/Safety-Drive-AI/](https://simwingachristopher0-ship-it.github.io/Safety-Drive-AI/)
2. Type your question in the box
3. Click `Send`

### Example Questions:
- `Business ideas I can start with 2000 kwacha in Lusaka`
- `How to grow maize in rainy season`
- `Explain photosynthesis for grade 9`
- `Write a CV for me`
